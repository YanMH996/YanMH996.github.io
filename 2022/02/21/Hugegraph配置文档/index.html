HugeGraph开发文档

studio新建并切换图库

修改gremlin-server.yaml

添加
hugegraph/conf/gremlin-server.yaml
 graphs: {
   hugegraph: conf/hugegraph.properties,
   hugegraphfile: conf/hugegraphfile.properties,
   新图库名字: conf/新图库名字.properties
 }

修改rest-server.properties

 graphs=[
 hugegraph:conf/hugegraph.properties,
 hugegraphfile:conf/hugegraphfile.properties,
 新图库名字:conf/新图库名字.properties
 ]

新图库名字.properties

cp hugegraph.properties 新图库名字.properties
vi 新图库名字.properties
backend=hbase
serializer=hbase
store=新图库名字
# hbase backend config
hbase.hosts=10.255.175.xx
hbase.port=2181
保存并退出

初始化数据库

bin/init-store.sh
重启server
bin/stop-hugegraph.sh
bin/start-hugegraph.sh
可视化切图
hugegraph-studio-0.8.0/conf 下:
修改 studio的hugegraph-studio.properties
graph.name=hugegraph // hugegraph是想看的图库名
重启studio
